📥 安裝指南

以下是在 macOS 上安裝 OpenClaw 的完整流程。

📋 系統需求

💻

macOS

12.0 Monterey 或更新版本

🧠

記憶體

建議 8GB 以上（16GB 更佳）

💾

儲存空間

至少 5GB 可用空間

🔧

依賴項目

Homebrew、Node.js 18+

安裝 Homebrew（如果尚未安裝）

Homebrew 是 macOS 上最受歡迎的套件管理器。

# 複製並貼上到終端機執行
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安裝 Node.js

OpenClaw 需要 Node.js 18 或更新版本。

brew install node

下載並安裝 OpenClaw

從官方網站下載最新版本的 OpenClaw 安裝包。

brew install --cask openclaw
# 或從官網手動下載：https://openclaw.ai/download

首次啟動與設定

第一次啟動時，OpenClaw 會引導你完成初始設定。

open -a OpenClaw
# 或在 Applications 資料夾中點擊 OpenClaw.app

設定過程中你可以：
• 選擇喜歡的 AI 模型
• 設定偏好設定
• 連結你的通訊平台（Telegram、Discord 等）

驗證安裝

完成設定後，在對話框中輸入任意內容，確認 OpenClaw 正常運作。

你好，OpenClaw！
# 應該會收到回應

💡 實用建議

充分發揮 OpenClaw 潛力的最佳實踐。

  • 🧠

    選擇適合的 AI 模型

    如果注重速度，選擇 MiniMax 或 Claude；
    如果注重品質，選擇 GPT-4 或 Gemini Pro；
    如果完全離線需求，使用 Ollama + Llama 3。

  • 🔗

    連結多個通訊平台

    OpenClaw 支援 Telegram、Discord、WhatsApp 等平台。
    建議根據用途分開使用，例如：工作用 Discord、個人用 Telegram。

  • ⚙️

    善用系統提示詞

    透過調整 SOUL.md 和 USER.md，
    讓 OpenClaw 更了解你的需求和偏好，提升回答品質。

  • 🔒

    設定 API 配額上限

    使用雲端 API 時，建議在供應商後台設定每日用量上限，
    避免意外費用產生。

  • 📁

    定期備份設定

    你的 workspace 包含所有設定和記憶，
    建議使用 Git 或 iCloud 定期備份。
